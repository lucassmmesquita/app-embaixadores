# APP EMBAIXADORES — Correções e Melhorias

Olá, segue abaixo a lista de correções e melhorias necessárias para o **App Embaixadores**.

## 1. Materiais

- Validar com a equipe da campanha quais materiais serão disponibilizados no app.
- Remover as seções **Artigos** e **Docs**.
- Integrar a seção de **Posts** com o Instagram.
- Implementar o compartilhamento de materiais seguindo a mesma lógica e fluxo do convite do app.

### Fluxo esperado

Cada material deve possuir uma landing page própria.

O link compartilhado deve conter:

- Código do material;
- Código do usuário que compartilhou;
- Link com o código de convite do usuário.

### Pontuação

- Quando uma pessoa clicar no material compartilhado, o usuário que compartilhou ganha **10 pontos**.
- Caso a pessoa também realize o cadastro no app, o usuário que compartilhou ganha mais **10 pontos**.

---

## 2. Avisos

- Remover os dados mockados.
- Adicionar a funcionalidade de **limpar avisos**.
- Integrar os avisos com o Admin, permitindo o cadastro e gerenciamento dos avisos pelo painel administrativo.

---

## 3. Badges / Minhas Conquistas

- As badges devem representar os perfis que o usuário alcança conforme evolui na jornada.
- Deixar mais claro quais são os perfis disponíveis e quais metas de pontuação são necessárias para alcançar cada um.
- Apresentar a jornada de pontuação de forma visual e compreensível para o usuário.

---

## 4. Missões

- Reorganizar as missões por tipo.
- Manter apenas os seguintes tipos de missão:
  - **Convites aceitos / Engajamento**;
  - **Participação em eventos**;
  - **Compartilhamento de materiais**.
- Remover os demais tipos de missão já existentes que não se encaixem nessa nova estrutura.

---

## 5. Eventos / Agenda

- Permitir o cadastro de eventos pelo Admin.
- Implementar o convite para eventos seguindo a mesma lógica e fluxo do convite do app.

### Fluxo esperado

Cada evento deve possuir uma landing page própria.

O link compartilhado deve conter:

- Código do evento;
- Código do usuário que compartilhou;
- Link com o código de convite do usuário.

### Pontuação

- Quando uma pessoa clicar no evento compartilhado, o usuário que compartilhou ganha **10 pontos**.
- Caso a pessoa também realize o cadastro no app, o usuário que compartilhou ganha mais **10 pontos**.

---

## Observação

A lógica de compartilhamento, rastreamento por código e pontuação deve ser padronizada entre **convites, materiais e eventos**, garantindo consistência no fluxo do usuário e no registro dos pontos.
